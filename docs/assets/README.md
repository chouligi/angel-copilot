# Docs Assets

This folder stores launch visuals used by the root README and release pages.

## Recommended files

- `single_deal_md_example.png`
  - Screenshot of a single-deal memo response.
- `batch_report_example_1.png`
  - Screenshot of comparative batch report overview (ranked summary).
- `batch_report_example_2.png`
  - Screenshot of individual-deal section in the batch report.
- `dealflow_triage_demo_front_cover.png`
  - Source front cover image for Dealflow Triage demo.
- `youtube_play_button.png`
  - Source YouTube play button asset used on the demo thumbnail.
- `dealflow_triage_demo_front_cover_play.png`
  - Clickable README thumbnail with centered play-button overlay.

## Guidelines

- Use synthetic or anonymized deal data only.
- Prefer PNG for screenshots and GIF/MP4 for short demos.
- Keep file names stable so README links do not need updates.
- Recommended width: 1400-2000px for crisp GitHub rendering.

## Status

Assets currently use synthetic sample outputs from this repository.
